# **App Name**: Global Petrotech

## Core Features:

- Comprehensive Company Overview: Display a detailed profile including focus areas, strengths, experience, and core sectors (Oil & Gas, Power, Energy, Infrastructure).
- History Timeline: Showcase Global Petrotech's journey from foundation to present day, highlighting key milestones and regional recognition through an animated timeline.
- Vision and Mission Display: Clearly present the company's vision and mission statements, emphasizing quality, innovation, safety, and integrity.
- Interactive Organizational Chart: Provide an interactive organizational chart showcasing leadership roles (CEO, General Manager, etc.) and typical project team structure with role descriptions.
- Service Showcase: Exhibit core services such as civil & mechanical construction, flowlines & hook-up works, structural fabrication, EPC project support, and pipeline installation using illustrative icons and brief descriptions.
- Project Portfolio Gallery: Create an image gallery of major projects completed with filters for different sectors, clients, and statuses. Use animations on image hover to display project details such as client, contractor and completion date. An LLM tool generates a summary of project success factors and challenges, incorporating available metadata when it deems relevant.
- Contact Form: Implement a contact form for inquiries with fields for name, email, subject, and message, ensuring direct communication with Global Petrotech.

## Style Guidelines:

- Primary color: Deep eggplant (#572649) to reflect reliability and industrial expertise, as specified by the user.
- Background color: Very light gray (#FFFFFF) to maintain a clean and professional look. It's a desaturated version of the primary hue to support a light color scheme.
- Accent color: Soft lavender (#805773) as an analogous hue to the primary, adding depth to the color scheme and used to highlight interactive elements.
- Body font: 'PT Sans', a humanist sans-serif, for clear and readable body text throughout the site.
- Headline font: 'Playfair', a modern serif font similar to Didot, used for headings to provide an elegant and high-end feel.
- Use clean, professional icons related to each service area, ensuring visual clarity and ease of navigation. The icons should be designed in a flat style.
- Employ a responsive, grid-based layout ensuring adaptability across devices (desktop, tablet, mobile) and maintaining a consistent visual structure.
- Incorporate subtle animations on scrolling and hovering to enhance user experience without overwhelming the design, adding a dynamic feel to the website.
